There is nothing I love more than creating new things through code. It is satisfying to imagine things that will make people's life easier. It is something priceless and I am convinced that the best way to create amazing software is with an enthusiastic and entrepreneurial team that is not afraid to evolve within improvement.

Web Developer from the heart and Desktop Developer by profession, I have 5 years of experience developing software for companies in different sectors: Financial, Educational, POS in general and Research development.

I'm a "New Web Technology Enthusiastic", I love learning new things as well analyzing whether they can be used professionally or not. Lately I have been following very closely the progress and maturity of MEAN Stack Development (Mongo Express Angular NodeJS), in which I have begun to develop a project to manage talent and models for some friends modeling agencies.

Once in a while, I help voluntarily non-governmental associations and college asociated projects. I find myself currently helping www.movie-nights.ca as well getting requirements for the website.

The software I have worked in recent years has been primarily developed with some of these stacks:

  1. PHP (POO) + MySQL
  2. PHP (Yii Framework) + MySQL
  3. PHP (Yii Framework) + WCF Services + SQL SERVER
  4. C# (Windows Forms) + WCF Services + SQL SERVER
  5. ASP.NET MVC + LINQ + SQL SERVER
  6. Android + Yii Framework Services
  7. Android + WCF Services
  8. MEAN (Mongo Express Angular Node)
